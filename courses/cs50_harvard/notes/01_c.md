## üéØ Key Concepts

### From Scratch to C

**What it is:**  
Translating Scratch's visual blocks into text-based C code. Same concepts, different syntax.

**Why it matters:**  
Understanding this mapping helps you see that programming fundamentals are universal - only syntax changes.

**Scratch ‚Üí C mapping:**
- `say "hello"` ‚Üí `printf("hello\n");`
- `set counter to 0` ‚Üí `int counter = 0;`
- `change counter by 1` ‚Üí `counter++;` or `counter = counter + 1;`
- `forever` ‚Üí `while (true)`
- `repeat 10` ‚Üí `for (int i = 0; i < 10; i++)`
- `if <touching edge?>` ‚Üí `if (x > width || x < 0)`

### C Basics

**What it is:**  
A compiled, procedural programming language created in 1972. Low-level, close to hardware, very fast.

**Why it matters:**  
Understanding C teaches how computers work at a fundamental level. Many modern languages (C++, Java, JavaScript) borrowed C's syntax.

**Key points:**
- **Compiled language:** Source code ‚Üí compiler ‚Üí machine code
- **Statically typed:** Must declare variable types
- **Manual memory management:** Programmer controls memory (unlike Python)
- **Fast execution:** Compiles to native machine code
- **Semicolons required:** End every statement with `;`
- **Case-sensitive:** `variable` ‚â† `Variable`
- **main() function:** Every program starts here

---

### Compilation Process

**What it is:**  
Converting human-readable source code into machine-executable binary code.

**Why it matters:**  
Understanding compilation helps debug errors and optimize performance.

**Steps:**
1. **Preprocessing:** Handle `#include`, `#define` directives
2. **Compiling:** Convert C to assembly language
3. **Assembling:** Convert assembly to machine code (0s and 1s)
4. **Linking:** Combine with libraries to create final executable

**Command:** `make hello` or `clang -o hello hello.c`

---

### Data Types

**What it is:**  
Different categories of data with different sizes and capabilities.

**Why it matters:**  
Choosing correct type prevents errors and saves memory.

**Basic types:**
- **int:** Integers (-2 billion to +2 billion)
- **long:** Larger integers
- **float:** Decimal numbers (6-7 digits precision)
- **double:** Double precision decimals (15-16 digits)
- **char:** Single character ('a', 'Z', '7')
- **bool:** True or false (requires `#include <stdbool.h>`)
- **string:** Text (requires `#include <cs50.h>` in CS50)

**Format specifiers:**
- `%i` or `%d` for int
- `%f` for float/double
- `%c` for char
- `%s` for string
- `%li` for long

---

### Operators

**What it is:**  
Symbols that perform operations on values.

**Arithmetic:**
- `+` addition
- `-` subtraction
- `*` multiplication
- `/` division
- `%` modulo (remainder)

**Assignment:**
- `=` assign
- `+=` add and assign (`x += 5` same as `x = x + 5`)
- `++` increment by 1
- `--` decrement by 1

**Comparison:**
- `==` equal to
- `!=` not equal
- `<` less than
- `>` greater than
- `<=` less than or equal
- `>=` greater than or equal

**Logical:**
- `&&` and
- `||` or
- `!` not

---

### Conditions

**What it is:**  
Making decisions in code based on boolean expressions.

**Syntax:**
```c
if (condition)
{
    // code if true
}
else if (another_condition)
{
    // code if first false, this true
}
else
{
    // code if all false
}
```

**Key points:**
- Use curly braces `{}` for code blocks
- Condition must be in parentheses
- Can nest conditions
- Boolean expressions evaluate to 1 (true) or 0 (false)

---

### Loops

**What it is:**  
Repeating code multiple times.

**While loop:**
```c
while (condition)
{
    // repeat while condition is true
}
```

**For loop:**
```c
for (initialization; condition; update)
{
    // repeat
}
```

**Do-while loop:**
```c
do
{
    // execute at least once
}
while (condition);
```

**Key points:**
- For loops best for known iterations
- While loops best for unknown iterations
- Do-while guarantees at least one execution
- Watch for infinite loops!

## üíª Code Examples

### Hello World
```c
#include <stdio.h>

int main(void)
{
    printf("hello, world\n");
    return 0;
}
```

**Output:**
```
hello, world
```

**Explanation:**
- `#include <stdio.h>` - Import standard I/O library
- `int main(void)` - Entry point of program
- `printf()` - Print formatted text
- `\n` - Newline character
- `return 0` - Exit with success code

### Variables and Input

Hello, Manuel! You are 25 years old.

### While Loop
```c
#include <stdio.h>

int main(void)
{
    int i = 0;
    while (i < 5)
    {
        printf("%i\n", i);
        i++;
    }
    return 0;
}
```

**Output:**
```
0
1
2
3
4
```

---

### For Loop
```c
#include <stdio.h>

int main(void)
{
    // Print 0-4
    for (int i = 0; i < 5; i++)
    {
        printf("%i\n", i);
    }
    
    // Print hash marks
    for (int i = 0; i < 10; i++)
    {
        printf("#");
    }
    printf("\n");
    
    return 0;
}
```

**Output:**
```
0
1
2
3
4
##########
```

### Do-While (Input Validation)
```c
#include <cs50.h>
#include <stdio.h>

int main(void)
{
    int n;
    do
    {
        n = get_int("Height: ");
    }
    while (n < 1 || n > 8);
    
    printf("You entered: %i\n", n);
    return 0;
}
```

**What it does:** Keeps asking until valid input (1-8)

---

### Mario Pyramid
```c
#include <cs50.h>
#include <stdio.h>

int main(void)
{
    int height = get_int("Height: ");
    
    // For each row
    for (int i = 0; i < height; i++)
    {
        // Print spaces
        for (int j = 0; j < height - i - 1; j++)
        {
            printf(" ");
        }
        
        // Print hashes
        for (int j = 0; j <= i; j++)
        {
            printf("#");
        }
        
        printf("\n");
    }
    
    return 0;
}
```

**Output (height = 4):**
```
   #
  ##
 ###
####
```

## üìñ Important Terms

| Term | Definition | Example |
|------|------------|---------|
| **Compiler** | Converts source code to machine code | clang, gcc |
| **Syntax** | Grammar rules of language | `printf("hello");` |
| **Variable** | Named storage location | `int x = 5;` |
| **Data type** | Category of data | int, float, char |
| **Function** | Reusable block of code | `main()`, `printf()` |
| **Library** | Collection of pre-written functions | stdio.h, cs50.h |
| **Argument** | Value passed to function | In `printf("hi")`, "hi" is argument |
| **Return value** | Value function gives back | `main()` returns 0 |
| **Format specifier** | Placeholder in printf | `%i`, `%s`, `%f` |
| **Header file** | Contains function declarations | .h files |
| **Semicolon** | Statement terminator | Required at end of statements |
| **Curly braces** | Define code blocks | `{ }` |

## üí° Key Takeaways

1. **C teaches fundamentals** - Understanding C helps with all languages
2. **Compilation catches errors early** - Unlike interpreted languages
3. **Types matter** - Must declare types, prevents many errors
4. **Semicolons are mandatory** - Forgetting them is #1 beginner mistake
5. **Curly braces define scope** - Always use them for clarity
6. **For loops are powerful** - Master them, use them everywhere
7. **printf debugging works** - Add `printf` statements to see what's happening
8. **CS50 library helps** - `get_int()`, `get_string()` handle input validation
9. **Start simple, test often** - Build incrementally, compile frequently
