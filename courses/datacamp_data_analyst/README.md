## Notebooks
Each course has a dedicated notebook in `/notebooks` folder following the format `XX_course_name.ipynb`.

**Notebook Structure:**
- Course exercises and experiments
- Key concepts and syntax examples
- Personal notes and discoveries
- "What I Learned" sections
- Code snippets for future reference

**Naming Convention:**
- **01_intro_to_python.ipynb** - Python basics, variables, data types, lists
- **02_intermediate_python.ipynb** - Matplotlib basics, dictionaries, pandas intro
- **03_data_manipulation_pandas.ipynb** - DataFrames, filtering, aggregating
- **04_joining_data_pandas.ipynb** - Merging, joining, concatenating
- **05_intro_statistics_python.ipynb** - Random numbers, distributions, probability
- **06_data_viz_matplotlib.ipynb** - Creating plots, customization
- **07_data_viz_seaborn.ipynb** - Statistical visualizations
- **08_exploratory_data_analysis.ipynb** - Real-world data exploration
- **09_working_dates_times.ipynb** - DateTime manipulation
- **10_writing_functions.ipynb** - Function design, documentation
- **11_intro_importing_data.ipynb** - Reading files, APIs basics
- **12_intermediate_importing_data.ipynb** - Web scraping, APIs advanced
- **13_cleaning_data.ipynb** - Data quality, missing values, duplicates
- **14_categorical_data.ipynb** - Category dtypes, encoding
- **15_data_communication.ipynb** - Storytelling with data
